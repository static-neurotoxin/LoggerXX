const boost::format LogXX::message::getMessageHeader ( std::string  formatStr)

Get the raw boost format object for the header, usable for streaming These are the values that are available.

  • %1% Date
  • %2% Time
  • %3% Log level text
  • %4% Thread ID
  • %5% Filename
  • %6% Filename with full path
  • %7% Extended function name if defined, otherwise basic funtion name
  • %8% Basic function name
  • %9% Extended function name

The default format string is "%1% %2% %3% [%4%] %5% %7%"
